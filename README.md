# lambda-example
Example SAM templates to deploy AWS Lambda used by various labs / workshop (https://chatbotworkshop.com, https://sagreenlab.com, https://containersworkshop.com)

## Package REST API
```shell
# change into the rest-api service
cd rest-api
```

```shell
npm run package
```

## Deploy REST API

```shell
npm run deploy
```
